    1

    Locate your AddOns folder

    2

    Create the addon folder

    LicenseDemo
    3

    Move the downloaded files

    Put all 3 files into the LicenseDemo folder:

    • LicenseDemo.toc
    • Core.lua
    • Data.lua
    4

    Reload the game

    In WoW, type:

    /reload

    /ldemo
